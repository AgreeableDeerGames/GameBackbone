Detailed Description

template<class... Handlers>
class GB::InputRouter< Handlers >

Stores several GB::InputHandler instances and forwards handleEvent calls to each of them in order. Once the event is successfully handled, no other GB::InputHandler may handle the event.

Template Parameters
...HandlersA variadic list of GB::InputHandler that will receive the forwarded handleEvent calls. The priority of the Handlers is defined by the order that they were passed in. Earlier Handlers have higher priority.
