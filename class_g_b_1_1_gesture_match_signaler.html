Detailed Description

template<typename EventCompare, typename EventFilter, std::enable_if_t< is_event_comparator_v< EventCompare >, bool > = true, std::enable_if_t< is_event_filter_v< EventFilter >, bool > = true>
class GB::GestureMatchSignaler< EventCompare, EventFilter,, >

Signal class that fires an action when it handles a set of inputs that match an expected gesture.

Template Parameters
EventCompareThe type of the comparator that will be used to check if two events match.
EventFilterThe type of the filter that determines if an event should be matched against the expected gesture or skipped entirely.

enum GB::GestureMatchSignaler::MatchBehavior
strong

Indicates the desired behavior after a match is found.

Enumerator
Penultimate 

Signal with every subsequent event that matches the last event in the gesture.

Reset 

Puts the gesture back to position 0 after it completes and allows the gesture to complete again.

Block 

Stop handling events after signaling the next match.
